MYSQL datubāze ir savienota ar Intellij Java.
Java saņem datus no MYSQL un atspoguļo tos tabulā.
Tika izveidota jauna directory kurā tika ielikts JAR fails (kafijas krūzīte).
Java ļauj ievadīt datus MYSQL datubazē, ievietojot tos izmantojot SWING UI, kas ir izveidots GUI klasē.
DB logic klase ļauj savienoties ar MYSQL 
Main klase ļauj palaist aplikāciju Swing UI.
