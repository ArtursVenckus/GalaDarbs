MYSQL datubaze ir savienota ar Intellij java.
Ja dabū datus no MYSQL un atspoguļo tos tabulā.
Tikka izveidota jauna directory kur tikka ielikts JAR file (kafijas krūzīte).
Java ļauj ievadīt datus MYSQLdatubazē ievietojot tos izmantojos SWING UI kas ir izveidots GUL klasē.
DB logic klase ļauj savienoties ar mySQL 
Main klase lauj palaist aplikāciju SWING UI.
